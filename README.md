# 终端弹幕酱

最近在 Discover 里找到了一个下拉终端软件 Yakuake ，感觉很好用，然后就想做个配合这个软件用的东西，正好看到有人做了一个终端里的弹幕姬，于是自己也做了一个，以后发弹幕就不用切到浏览器了，直接快捷键按出终端打字就行了  
  
最大的收获就是体验了一遍 C# 里操作二进制数据, 以及在 C# 里实现了一遍祖传弹幕接收器

## 用法
`dotnet run config.json [房间号]`

## 配置文件格式
```
{
    "roomId": 默认房间号,
    "sessdata": "从cookie里找",
    "csrf": "从cookie里找",
    "buvid3": "从cookie里找",
    "uid": 你的uid
}
```
